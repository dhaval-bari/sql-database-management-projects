# SQL Database Management Projects

This repository contains 10 structured SQL mini-projects based on real-world systems. Each project demonstrates database design, data manipulation, and complex query execution using MySQL.

## Projects Included

1. Online Bookstore
2. Hospital Management System
3. University Management System
4. Airline Reservation System
5. Hotel Management System
6. Library Management System
7. Inventory Management System
8. Online Food Delivery System
9. Cinema Ticket Booking System
10. E-Learning Platform

## Tech Stack

* MySQL
* MySQL Workbench

## Key Concepts Covered

* Database Design (ER-based thinking)
* DDL & DML operations
* Joins (INNER, LEFT)
* Aggregations (SUM, COUNT, AVG)
* Subqueries
* Group By & Having
* Real-world case-based queries

## Structure

Each project folder contains:

* `schema.sql` → Table creation + data insertion
* `queries.sql` → 25 SQL queries
* `screenshots/` → Execution proof

## How to Run

1. Open MySQL Workbench
2. Run `schema.sql`
3. Execute `queries.sql`
4. View results

## Purpose

This project was developed as part of a Database Management Systems curriculum to strengthen practical SQL skills and real-world database modeling.
